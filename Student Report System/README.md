# 🧠 Mini Project: Student Information System

Create a small program that:

### Requirements

1. Ask the user for:

   * Name
   * Age
   * Marks in 3 subjects

2. Use **typecasting** to convert marks to numbers.

3. Calculate:

   * Total marks
   * Average marks

4. Display result like this:

```
---- Student Report ----
Name: Rahul
Age: 20
Total Marks: 240
Average: 80.0
Passed: True
```

Rule:

```
If average >= 40 → Passed
Else → Failed
```
